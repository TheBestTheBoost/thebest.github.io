Электродинамика

Электростатика

Определение: Электростатика - раздел электродинамики, изучающий взаимодействие неподвижных электрических зарядов.

Закон Кулона: \[F = k \frac{|q_1 q_2|}{r^2}\]

Напряженность электрического поля: \[\mathbf{E} = \frac{\mathbf{F}}{q}\]

Напряженность поля точечного заряда: \[\mathbf{E} = \frac{k q}{r^2}\frac{\mathbf{r}}{r}\]

Потенциал электрического поля: \[\phi = \frac{k q}{r}\]

Связь напряженности и потенциала: \[\mathbf{E} = -\nabla \phi\]

Энергия взаимодействия зарядов: \[W = \frac{k q_1 q_2}{r}\]

Энергия конденсатора: \[W = \frac{C U^2}{2} = \frac{qU}{2} = \frac{q^2}{2C}\]

Электроемкость: \[C=\frac{q}{U}\]

Постоянный ток

Определение: Постоянный ток - это электрический ток, не меняющийся со временем.

Сила тока: \[I = \frac{q}{t}\]

Закон Ома для участка цепи: \[I = \frac{U}{R}\]

Закон Ома для полной цепи: \[I = \frac{\varepsilon}{R + r}\]

Сопротивление проводника: \[R = \rho \frac{l}{S}\]

Мощность электрического тока: \[P = UI = I^2R = \frac{U^2}{R}\]

Работа электрического тока: \[A = UIt\]

Закон Джоуля-Ленца: \[Q = I^2 R t\]

Магнетизм

Определение: Магнетизм - это раздел электродинамики, изучающий взаимодействие движущихся электрических зарядов.

Сила Лоренца: \[\mathbf{F_L} = q [\mathbf{v} \times \mathbf{B}]\]

Модуль силы Лоренца: \[F_L = q v B \sin\alpha\]

Сила Ампера: \[\mathbf{F_A} = I [\mathbf{l} \times \mathbf{B}]\]

Модуль силы Ампера: \[F_A = I l B \sin\alpha\]

Магнитная индукция: \[B\]

Магнитный поток: \[\Phi = BScos\alpha\]

Закон электромагнитной индукции: \[\varepsilon = -\frac{\Delta\Phi}{\Delta t}\]

Индуктивность катушки: \[L = \frac{\Phi}{I}\]

Энергия магнитного поля катушки: \[W = \frac{LI^2}{2}\]
